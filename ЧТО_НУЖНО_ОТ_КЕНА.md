## 1. Supabase (база данных) — 10 минут, бесплатно

1. https://supabase.com → Sign Up (можно через GitHub).
2. New project → имя `margo-karat`, регион EU (Frankfurt), придумайте пароль базы.
3. Когда проект создастся: **SQL Editor → New query** → вставьте содержимое
   файла `supabase_schema.sql` → **Run**.
4. **Storage → New bucket**: `uploads` (private) и `avatars` (public).
5. **Authentication → Users → Add user**: создайте пользователя с вашим email
   (odessaradio3@gmail.com) и паролем — это вход в админ-панель.
6. Пришлите мне из **Settings → API**:
   - Project URL
   - anon (public) key
   - service_role key ⚠️ секретный — только мне в чат, никуда больше

## 2. Stripe (приём платежей) — 20–30 минут

1. https://stripe.com → Create account → страна вашего бизнеса.
2. Пройдите верификацию (данные, банковский счёт для вывода — IBAN).
3. Пришлите мне из **Developers → API keys**: Secret key (`sk_live_...`).
   Вывод денег: Stripe сам переводит на ваш счёт (Settings → Payouts, по умолчанию ежедневно).

## 3. Почта Маргариты — 10 минут

1. Создайте Gmail, например `margokarat.official@gmail.com`.
2. Включите двухфакторную защиту → https://myaccount.google.com/apppasswords →
   создайте «Пароль приложения».
3. Пришлите мне адрес и этот 16-значный пароль приложения.

## 4. Сервер 24/7 (бот + AI-агент) — 30 минут, бесплатно

Oracle Cloud Free Tier: https://signup.oraclecloud.com
(карта нужна для проверки личности, деньги не списываются).
После регистрации скажите мне — проведу по созданию сервера по шагам,
там же запустим `margo_bot.py` и `margo_agent.py` (инструкция уже в
`МАРГО_БОТ_ИНСТРУКЦИЯ.md`).

## 5. Хостинг сайта (когда будет домен)

Купите домен (например margokarat.com — Namecheap/GoDaddy, ~10 €/год).
Сайт можно бесплатно включить на GitHub Pages прямо из репозитория — скажите, включу.
